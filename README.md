# Northvalley Intelligence Website

Next.js website for Northvalley Intelligence LLC.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- Prettier
- Vercel-ready deployment
- Cloudflare DNS-ready domain setup

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Deployment

The preferred deployment target is Vercel. Cloudflare should manage DNS for `northvalleyintel.com`.

See [docs/deployment.md](docs/deployment.md).
